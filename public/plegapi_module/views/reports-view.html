/plegapi/v1/reports

GET

Sending a get request to this API endpoint returns an array of JSON object all the reports available on that resource, with each array element(i.e a report) in JSON format or it returns an empty array if no report is available on the resource.

Example
GET http://www.example.com/plegapi/v1/reports
RETURNS array of reports [{report1},{report2},{report3}....] or an empty array [].

POST

Sending a post request to this API endpoint(mostly from a form field), is a request to create a new report. This request must be authorized, hence, the form field must contain parameters, which will also include an api_id and api_key field for authorization. If this request is successful, it returns the newly created report to the request sender, in JSON format.
REQUIRED PARAMETERS => title,state_id,report_time,mobile_user_id,sector_id and also api_key and api_id(this is compulsory for authorization);
OPTIONAL PARAMETERS => description,location_id

Example
POST http://www.example.com/plegapi/v1/reports
RETURNS a JSON object of the created report.

/plegapi/v1/reports/{id}

GET

Sending a get request to this API endpoint returns an object containing a report with the id supplied or an empty object if such report is non-existent.

Example
GET http://www.example.com/plegapi/v1/reports/2
RETURNS an object in the form {"report_id":2,"title":"*****","state_id":"****"} or an empty object {}.
