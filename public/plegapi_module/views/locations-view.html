/plegapi/v1/states

GET

Sending a get request to this API endpoint returns an array of JSON objects of all the states available on that resource, with each array element(i.e a report) in JSON format or it returns an empty array if no state is available on the resource.

Example
GET http://www.example.com/plegapi/v1/states
RETURNS array of states [{state1},{state2},{state3}....] or an empty array [].

/plegapi/v1/states/{id}/lgas

GET

Sending a get request to this API endpoint returns an array of JSON objects of all the lgas under the state having the id supplied in the url, with each array element(i.e an lga) in JSON format or it returns an empty array if no state is available on the resource or if the states exists with no associated lga.

Example
GET http://www.example.com/plegapi/v1/states/2/lgas
RETURNS array of lgas under the state with id of 2 [{lga1},{lga2},{lga3}....] or an empty array [].

/plegapi/v1/states/{id}

GET

Sending a get request to this API endpoint returns an object containing a state with the id supplied or an empty object if such report is non-existent.

Example
GET http://www.example.com/plegapi/v1/states/2
RETURNS an object in the form {"state_id":"****".....} or an empty object {}.

/plegapi/v1/lgas/{id}

GET

Sending a get request to this API endpoint returns an object containing an lga with the id supplied or an empty object if such lga is non-existent.

Example
GET http://www.example.com/plegapi/v1/lgas/2
RETURNS an object in the form {"lag_id":2,"*****":"*****","state_id":"****"} or an empty object {}.
