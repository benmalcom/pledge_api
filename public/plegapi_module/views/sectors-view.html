/plegapi/v1/sectors

GET

Sending a get request to this API endpoint returns an array of JSON objects of all the sectors available on that resource, with each array element(i.e a report) in JSON format or it returns an empty array if no sector is available on the resource.

Example
GET http://www.example.com/plegapi/v1/sectors
RETURNS array of sectors [{sector1},{sector2},{sector3}....] or an empty array [].

/plegapi/v1/sectors/{id}

GET

Sending a get request to this API endpoint returns an object containing a sector with the id supplied or an empty object if such report is non-existent.

Example
GET http://www.example.com/plegapi/v1/sectors/2
RETURNS an object in the form {"sector_id":2,"name":"*****"} or an empty object {}.
